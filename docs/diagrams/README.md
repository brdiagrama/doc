# Diagramas

Esta pasta contém todos os diagramas UML e outros diagramas técnicos do projeto.

## Conteúdo

### Diagramas de Caso de Uso
Arquivos que começam com `caso-uso-*` mostram as interações entre usuários e sistema.

### Diagramas de Classe
Arquivos que começam com `diagrama-classe-*` representam a estrutura das classes do sistema.

### Outros Diagramas
- **Sequência**: fluxo de interações entre objetos ao longo do tempo
- **Atividades**: fluxo de trabalho ou processos
- **Estados**: diferentes estados de um objeto e suas transições

## Formatos Aceitos

- `.png`, `.jpg`, `.svg` - Imagens exportadas dos diagramas
- `.drawio` - Arquivos editáveis do Draw.io
- `.puml` - Arquivos PlantUML (texto)
- `.mdj` - Arquivos StarUML

## Como Adicionar um Diagrama

1. Crie o diagrama usando sua ferramenta preferida
2. Exporte como imagem (PNG recomendado para qualidade)
3. Se possível, salve também o arquivo fonte editável
4. Use nomenclatura descritiva: `tipo-nome-descritivo-v1.png`

Exemplo:
```
caso-uso-sistema-vendas-v1.png
caso-uso-sistema-vendas-v1.drawio
```
